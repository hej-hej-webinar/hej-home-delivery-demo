# Wrap runbook, 5-minute close and CTAs

## Goal
Close the session in five minutes with clear calls to action for teams and leaders.

## Presenter flow
1. Recap the storyline, old world habits, modern risks, and layered controls.
2. Summarise segment outcomes, data hygiene, supply chain controls, code security, and governed AI use.
3. Give practical next steps for the first 30 days.
4. End with concrete CTAs for implementation ownership.

## Call to action
- Adopt policy-backed dependency controls this week.
- Baseline code scanning and triage with accountable owners.
- Formalise Copilot usage patterns, including plan mode and adversarial review.
- Treat workflow security as part of product security, not platform overhead.
