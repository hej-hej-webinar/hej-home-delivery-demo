# Plan mode and adversarial challenge prompts

## Initial plan prompt

Plan the implementation of a delivery-preferences CSV import and customer lookup helper for this TypeScript service. Do not change files yet. Include tests and a pull request outline.

## Adversarial challenge prompt

Challenge the plan before implementation. Look only for meaningful risks and missing assumptions in these areas:

- Personal data or realistic fixture data.
- New dependencies and package install behavior.
- Malformed input and abuse cases.
- CodeQL-detectable vulnerabilities.
- CI permissions, action pinning, and secrets exposure.

Return the top five concerns and a revised implementation checklist.
